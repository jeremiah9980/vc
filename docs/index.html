GitHub-ready sports media workflow

DriveClip Pipeline

Clean up sports clips automatically from Google Drive using repeatable presets, FFmpeg processing, and a simple publishing workflow.

Read the README View architecture

1. Ingest

Upload clips to Google Drive, mirror or sync them locally, and funnel them into a clean processing queue.

2. Process

Apply color cleanup, resize, audio normalization, and sport-specific presets using FFmpeg.

3. Publish

Review cleaned clips, send them into your editing app, and keep the project documentation live on GitHub Pages.
